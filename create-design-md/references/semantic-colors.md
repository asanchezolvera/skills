# Standardized Semantic Color Palettes

All Golden Hippo brands use the same semantic color palettes, derived from Tailwind CSS defaults. These are NOT customizable per-brand — they ship as-is in every DESIGN.md.

## Error / Danger (Tailwind Red)

| Step | Hex     |
| ---- | ------- |
| 50   | #fef2f2 |
| 100  | #fee2e2 |
| 200  | #fecaca |
| 300  | #fca5a5 |
| 400  | #f87171 |
| 500  | #ef4444 |
| 600  | #dc2626 |
| 700  | #b91c1c |
| 800  | #991b1b |
| 900  | #7f1d1d |
| 950  | #450a0a |

**Typical mappings:**

- Error text: 600 (light surfaces), 400 (dark surfaces)
- Error background: 50 (light), 950 (dark)
- Error border: 200 (light), 800 (dark)
- Destructive button: 600 background, white text
