# Tailwind CSS Default Neutral Color Palettes

These are the nine default neutral palettes from Tailwind CSS (v4.2+). Each palette runs from 50 (lightest) to 950 (darkest). When the user selects a neutral palette, use these exact hex values.

The first five — Stone, Gray, Neutral, Zinc, Slate — are the classic neutrals. The remaining four — Taupe, Mauve, Mist, Olive — were added in v4.2 as "tinted neutrals": their chroma peaks around 0.021–0.034 (barely above zero), which keeps them firmly neutral while giving each a distinct hue character. Hex values are the nearest sRGB approximation of the canonical OKLCH definitions.

## Palette Selection Guide

When helping users choose, consider the brand's primary colors and overall character:

**Classic neutrals** (lower commitment, more flexibility):

- **Warm brand colors** (reds, oranges, yellows, terracotta) → Stone or Neutral
- **Cool brand colors** (blues, purples, teals) → Slate or Zinc
- **Mixed/vibrant brand colors** → Neutral or Gray
- **Tech/modern brands** → Zinc or Slate
- **Minimal/invisible neutrals** → Neutral

**Tinted neutrals** (more character, more opinion):

- **Warm refined brands** (gallery, fashion, hospitality) → Taupe
- **Editorial / moody / purple-leaning brands** → Mauve
- **Healthcare, wellness, productivity, ambient** → Mist
- **Earthy / natural / muted brands** → Olive

The tinted neutrals work best when they lean _with_ the brand colors rather than against them. Olive against a saturated cool blue brand would feel mismatched; Mist against the same brand reinforces the cool palette. When in doubt, lean classic.
